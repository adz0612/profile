Jul 2018 – Nov 2018

Smart Wardrobe Management System

This project was aimed to create a fully portable and smart system that secures a wardrobe and also provides many other features. This IOT device can protects the key of a wardrobe instead of meddling around with the locking mechanism. It logs status, analog data to the cloud gives smart clothing suggestions with respect to outside temperature, gives email alerts in case of a security breach and send a buy link regarding the clothing suggestion. This entire system is further accompanied by a companion Android app It was ranked third in the VIT-AP Engineering Expo.

Jul 2019 – Nov 2019

Brain Controlled Wheelchair

This project was aimed to help the paralyzed or physically challenged people to have better communication and mobility in the real world. The brain controlled wheelchair is designed to move in all four directions with respect to the signals received by the brain using a EEG machine.The wheelchair also features a keyboard layout which allows the user to communicate using brain signals and express their thoughts. It was ranked First in the VIT-AP Engineering Expo.
